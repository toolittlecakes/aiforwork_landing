# AI For Work — Landing

Лендинг практического курса «AI For Work»: ИИ-агенты (Codex, Claude Code) для не-разработчиков.

## Структура

- `index.html` — текущий рабочий вариант лендинга.
- `experiments/` — ранние черновики и стилевые эксперименты (разные направления дизайна, не используются).
- `лендинг_смыслы_2026-04-24.md` — бриф со смысловой структурой и tone of voice.

## Локальный просмотр

Это статический HTML без сборки. Достаточно открыть файл в браузере:

```bash
open index.html
```

Либо поднять простой сервер (удобнее для шрифтов и относительных путей):

```bash
python3 -m http.server 8000
# затем http://localhost:8000
```
