FAQ

Ответы на базовые вопросы перед участием.

Нужно ли уметь программировать?

Нет. Курс для не-разработчиков. Мы работаем с агентами через рабочие задачи.

Почему Codex / Claude Code?

Изначально это инструменты для программистов, но они применимы к файлам, таблицам, браузеру, отчётам и рабочим процессам.

А если Codex / Claude Code поменяют условия подписки?

Инструменты быстро меняются, поэтому курс строится вокруг навыка постановки задач, контекста и проверки результата.

Нужна ли платная подписка?

Да, нужна подписка ChatGPT Plus. Про варианты доступа расскажем в начале курса.

Ой, а можно без домашек?

Да, для этого есть базовый тариф: занятия, чат, записи и самостоятельная практика без проверки.

Можно ли прийти со своим кейсом?

Да. Финальный проект построен именно вокруг вашего реального рабочего процесса.

Если в вашей работе много ручной информационной рутины, часть её уже можно отдавать ИИ-агенту.

Покажем, как это делать на понятных рабочих задачах — без программирования и без магии.
